# CSV EDA Tool

A full-stack web application designed to perform automated Exploratory Data Analysis (EDA) on uploaded CSV files. This tool provides instant insights, visualizations, and statistical summaries through a clean web interface.

## 🚀 Features
- **File Upload:** Seamlessly upload CSV datasets.
- **Automated Summary:** View data types, missing values, and descriptive statistics.
- **Visualizations:** Generate charts and plots automatically (e.g., histograms, correlations).
- **Interactive UI:** A modern frontend for a better user experience.

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Frontend:** HTML, CSS, JavaScript (React/Vue if applicable)
- **Data Processing:** Pandas, NumPy
- **Visualization:** Matplotlib / Seaborn / Plotly
